NBA Dashboard and Player Statistics Predictive Model

Technical Scope: python (pandas, matplotlib), ML (Scikit-learn), Tableau
This project uses ETL methods to extract NBA data from an API, clean and transform the data into dataframes using python, and lastly export the desired datasets in CSV format. The mission with this project was to create interactive dashboards for coaches and players to analyze team and player statistics for upcoming games. The end goal is to develop an accurate predictive model for player stats and for a teams win probability vs specific opponents.
